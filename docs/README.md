# REGALARTE

Estado: PRE BUILD

## Descripción

REGALARTE es una plataforma de Discovery Commerce enfocada en regalos.

La experiencia principal no está centrada en catálogo tradicional.

Está centrada en:

- descubrimiento
- inspiración
- personalización
- gifting

Incluye:

- Ecommerce B2C
- Ecommerce B2B (Mayoristas)
- Universo Solística
- Gift Finder
- Gift Discovery
- Wishlist Emocional
- Caja Sorpresa
- Armá Tu Regalo
- PWA
